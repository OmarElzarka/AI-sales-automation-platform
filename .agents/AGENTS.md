## Technology Stack Constraints
- **.NET Version**: This project uses **.NET 10** (`net10.0`). Do not default to .NET 8 or .NET 9. When generating C# code, utilize C# 14 / .NET 10 features.
- **Angular Version**: This project uses **Angular 19**. Do not default to Angular 17 or 18. When generating frontend code, utilize the latest control flow syntax and standalone component paradigms introduced in Angular 17+.
